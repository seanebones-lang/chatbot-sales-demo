API Examples

Explore our API capabilities

BusinessChat API

Our powerful API allows developers to extend BusinessChat functionality and integrate with other systems.

Example: Creating a Custom Response

// Example using BusinessChat API to create a custom response
const BusinessChat = require('businesschat-api');

const client = new BusinessChat({
  apiKey: 'your-api-key',
  apiSecret: 'your-api-secret'
});

// Create a custom response handler
client.addResponseHandler('booking_request', (session) => {
  // Extract date and time from user message
  const datetime = extractDateTime(session.message);
  
  // Check availability
  if (isAvailable(datetime)) {
    return {
      type: 'text',
      content: `Great! I've reserved ${datetime} for your appointment.`
    };
  } else {
    return {
      type: 'text',
      content: `I'm sorry, ${datetime} is not available. Would you like to try another time?`
    };
  }
});

This is just one example of how our API can be used to create custom interactions. For full documentation, visit our developer portal.
